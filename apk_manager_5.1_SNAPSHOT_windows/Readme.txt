Requirements

Java 1.6

-------------------------------------------
Getting started on windows
-------------------------------------------
- Place apk in appropriate folder 
- Run script
- Minimize the script
- Edit files inside the project folder
- Maximize the script

-------------------------------------------
How to up APK-Manager
-------------------------------------------
put your framework-res.apk in /others
start a cmd in that folder and run :

java -jar apktool.jar if framework-res.apk

now you are ready to decompile/compile apks.
